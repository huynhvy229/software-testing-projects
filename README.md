# Hotel Management Testing Project

Dự án kiểm thử hệ thống Quản lý khách sạn (Team 2 người).

## Phần việc thực hiện (Vy)
Tôi chịu trách nhiệm chính kiểm thử các module cốt lõi sau:
* **Room Management**: Quản lý và thiết lập phòng.
* **Search & Booking**: Tìm kiếm và quy trình đặt phòng.
* **Payment**: Kiểm thử chức năng thanh toán của người dùng.

## Kết quả
* Hoàn thành thiết kế và thực thi Test Case với tỷ lệ pass cao ($95\% - 100\%$).
* Tổng hợp Bug Report và bàn giao tài liệu đúng tiến độ.

## Cấu trúc thư mục
* `/srs`: Tài liệu yêu cầu.
* `/test-cases`: File kịch bản kiểm thử.
